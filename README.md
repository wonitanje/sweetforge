# Sweetforge

## Quick start
```
npm instal
npm run build
npm start
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
#### Front-end 
```
npm run serve
```
#### Back-end 
```
npm run api
```

### Compiles and minifies for production
#### Front-end 
```
npm run build
```
#### Back-end 
```
npm start
```

### Or production with watch
#### Front-end ```npm run watch```
